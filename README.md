Schematics have been drawn using Eagle 7.5.

## Components
* [Single pole 16A relay with 5V coil](http://fr.farnell.com/oeg-te-connectivity/omih-ss-105lm-300/relais-spst-no-250vac-30vdc-16a/dp/1891727)
* [BD135 NPN bipolar transistor](http://fr.farnell.com/stmicroelectronics/bd135/transistor-npn-to-126/dp/9801529)
* [1N4007 rectifier diode](http://fr.farnell.com/multicomp/1n4007/diode-1a-1000v/dp/9565051)
* [MCP3428 I2C ADC](http://fr.farnell.com/microchip/mcp3428-e-sl/convertisseur-a-n-16-bits-i2c/dp/1825014)
* [100nF ceramic capacitor](http://fr.farnell.com/multicomp/mc0805b104k500a5-08mm/condensateur-mlcc-0-1uf-50v-x7r/dp/2395773)
* [10uF tantalum capacitor](http://fr.farnell.com/multicomp/mccb1e106m2ccb/condensateur-tantale-10uf-25v/dp/9708448)
* [AC/DC power supply](http://fr.farnell.com/xp-power/ecl15us05-e/psu-15w-5v-encaps-pcb-mount/dp/1435365)
* [39ohm resistor](http://fr.farnell.com/welwyn/mfr4-39rfi/resistance-0-5w-1-39r/dp/1099856)
* [82ohm resistor](http://fr.farnell.com/multicomp/mf25-82r/resistance-0-25w-1-82r/dp/9342303)
* [330ohm resistor](http://fr.farnell.com/welwyn/mfr3-330rfc/resistance-330r-1-0-4w-axial/dp/1565304)
* [560ohm resistor](http://fr.farnell.com/welwyn/mfr3-560rfc/resistance-0-4w-1-560r/dp/1565320)
* [Fuseholder](http://www.farnell.com/datasheets/2020897.pdf)
